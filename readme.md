## unix个人配置

[install](./install)目录为，个人unix装机目录，配置自动化，避免手动麻烦

如果很巧合， 装机的东西一样的话， 可以执行下列命令进行操作：

```bash
curl https://raw.githubusercontent.com/jiangtao/mydotfiles/master/install/boostrap.sh | sh
```
